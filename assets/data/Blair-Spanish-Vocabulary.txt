# front :: string, non-empty
#   - Gives the foreign translation of the text.
#   - When generating new cards, if using a sentance, then capitalize and punctuate correctly.
#
# back :: string, non-empty
#   - Gives the native translation of the text (English).
#   - When generating new cards, if using a sentance, then capitalize and punctuate correctly.
#
# emoji :: string, or empty.
#   - When available, represents a visual cue to aid in memorization.
#   - When generating new cards, only use emoji when for single word translation cards when there is a strong match.
#   - When generating new cards, never use an emoji if the card is a sentance or phrase.
#
# category :: string, non-empty, use verb tense names (e.g. Present, Imperfect, Preterit, ...).
#   - When generating new cards, capitalize the first character of each word in category.
#
# added :: date in YYYY-MM-DD form, or empty.
#   - When generating new cards, use the current local date for added.
#
# lastSeen :: date in YYYY-MM-DD form, or empty.
#   - When generating new cards, leave lastSeen empty.
#
# seen :: date in YYYY-MM-DD form, or empty.
#   - When generating new cards, use 1970-01-01 for seen.
#
# penalty :: integer
#   - Represents a score, that gets higher when the user ranks the card hard.
#   - Increases as the card is shown repeatedly.
#   - Increases faster if the card is ranked hard on each presentation.
#   - When generating new cards, leave penalty blank.
#
# level :: integer of (-1, 0, 1)
#   - Another representation of how hard a card is judged to me.
#   - A level of -1 means the user finds the card too easy, and only wants it presented when studying the 'review' set.
#   - A level of 0 means the user finds the card normally difficult, and wants it presented in the 'learning' sets.
#   - A level of 1 means the user finds the card very hard, and only wants it prested when studying the 'hard' set.
#   - When generating new cards, populate level as 1.
#
# comment :: string
#   - Useful for learning tips. 
#   - For example, explaining why on a given translation ser is used instead of estar.
#   - For example, explaining why on a given translation preterit is used instead of imperfect.
#
# Content :: Verb study
#   - This file is for learning conjugations of verbs in various tenses.
#   - It contains two types of cards: full congugation cards for a given verb and tense, and sentance cards.
#   - Sentance cards should prioritize a variety of sentance structures with different nouns, verbs, and other words.
#   - The primary goal of this study is to learn verb conjugations, but the secondary goal is to reinforce vocabulary.
#   - Choose these additional words from the users known vocabulary or occasionaly new words at a similar level.
#   - Prefer sentance structure that is natural in Spanish.
#   - Prefer sentances that are useful and someone may actually use.
#
# Convention :: for a verb full congugation card
#   - The foreign is all forms of the verb in a table like "[yo soy, ns. somos; tú eres, vs. sois; él es, ellos son]"
#   - The characters '[' and ']' denote the table start and end.
#   - Comma separates the cells in a table row.
#   - Semicolon separates the rows in a table.
#
